# 416WD-F2
## Product Landing Page Project

A project by Carlos Romeo D. C. of section CC14-2J in partial fulfilment of the course *CC14: Web Application Development* (with the RSO number `416WD`).

The assignment was to construct a product landing page for a specific product, complete with advantages and a hero image.

The company, locations, and chocolates depicted in the page are **completely fictional** and do not represent or resemble anything in the real world.

Development was made easier with the use of [Tailwind CSS](https://tailwindcss.com/). Four images total come from [Unsplash](https://unsplash.com); the rest are original works.
